## 🚀 Projects

### [CodeSnap](https://github.com/wronski04/codesnap)
Real-time collaborative code editor, think Google Docs for code snippets.

**Stack:** Next.js 15 · TypeScript · Supabase (Postgres + Realtime) · CodeMirror 6 · Tailwind CSS · Vercel

- Live multi-user editing via Supabase Realtime subscriptions
- Instant pad creation with custom or random slugs, no account needed
- Syntax highlighting for multiple languages via CodeMirror 6

### [Helpdesk System](https://github.com/wronski04/helpdesk)
University team project, a distributed helpdesk platform built with a microservices architecture.

**Stack:** .NET 9 · ASP.NET Core · PostgreSQL · RabbitMQ · MassTransit · Redis · Docker Compose · Ocelot API Gateway

- 5 independent services communicating via RabbitMQ events
- JWT authentication propagated through Ocelot gateway
- Containerized with Docker Compose for local orchestration

### [Commitly](https://github.com/wronski04/commitly) *(in progress)*
AI-powered changelog and release notes generator, paste your commits, get human-readable notes.

**Stack:** FastAPI · Python · PostgreSQL · SQLAlchemy · Alembic · React + Vite · TypeScript · Tailwind CSS · Groq LLM (llama-3.1-8b-instant) · Railway · Netlify

- LLM integration via Groq API with custom prompt engineering for two output tones: technical and user-friendly
- JWT authentication implemented manually with PyJWT + bcrypt
- Markdown export, per-project changelog history
